# CNN Handwritten Digit Classification - Infographic

## 📋 Project Overview

การจำแนกตัวเลขเขียนมือด้วย AI (CNN Handwritten Digit Classification)

สร้างปัญญาประดิษฐ์ (AI) ที่สามารถ "มอง" และ "แยกแยะ" ตัวเลข 0-9 จากลายมือคนได้ โดยเรียนรู้ผ่านชุดข้อมูล **MNIST** ซึ่งมีรูปภาพลายมือกว่า 60,000 รูป

## 🛠️ Tools & Technologies

- **Anaconda** - จัดการ Environment ป้องกัน Library ตีกัน
- **Ubuntu** - ระบบปฏิบัติการหลักที่เสถียร
- **PyTorch** - สร้างโครงข่ายประสาทเทียมระดับสากล
- **HTML & Tailwind CSS** - สร้างอินโฟกราฟิกนี้

## 🧠 CNN Algorithm Components

1. **Convolution** - สแกนหาจุดเด่นในภาพ (เส้นตรง ส่วนโค้ง ขอบตัวเลข)
2. **Pooling** - ลดขนาดภาพแต่เก็บข้อมูลที่สำคัญที่สุด
3. **Dropout (0.5)** - ป้องกันการ "ท่องจำ" เพื่อให้ทายลายมือใหม่ๆ ได้แม่นยำ

## ✨ Results

- **Accuracy**: 98-99% จากการเทรน 10 Epochs
- **Custom Image Testing**: ระบบรองรับการนำภาพลายมือที่วาดขึ้นใหม่มาตรวจสอบ
- **Image Processing**: แปลงภาพเป็นสีขาวดำ (Grayscale) ขนาด 28x28
